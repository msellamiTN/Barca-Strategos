# Barca-Strategos Logo Assets

## 🐘⚔️ Logo Concept: "Le Piège de Cannes"

The Barca-Strategos logo embodies Hannibal Barca's most famous military maneuver: the double envelopment at the Battle of Cannae.

## Logo Design Elements

### Primary Logo
- **File**: `barca-strategos-logo.svg`
- **Description**: Two bronze crescents facing each other with red center point
- **Usage**: Primary branding, website header, application icons

### Icon Variations
- **File**: `barca-strategos-icon.svg`
- **Description**: Minimalist crescent design for small sizes
- **Usage**: Favicons, mobile app icons, notifications

### Monochrome Logo
- **File**: `barca-strategos-mono.svg`
- **Description**: Single color version for security documents
- **Usage**: Technical documentation, watermarks, black & white printing

## Color Specifications

### Primary Colors
- **Rouge Carthaginois**: `#E2725B` (RGB: 226, 114, 91)
- **Bronze Antique**: `#CD7F32` (RGB: 205, 127, 50)
- **Noir Profond**: `#1A1A1A` (RGB: 26, 26, 26)

### Usage Guidelines
- **Minimum Size**: 24px height for digital use
- **Clear Space**: Equal to logo height on all sides
- **Background**: Always ensure contrast with background
- **Variations**: Use appropriate version for context

## Logo Symbolism

### The Two Crescents
- **Meaning**: Numidian cavalry wings encircling the enemy
- **Shape**: Dynamic, suggests movement and strategy
- **Color**: Bronze - historical authenticity and strength

### The Center Point
- **Meaning**: Critical asset under protection
- **Color**: Rouge Carthaginois - urgency and importance
- **Shape**: Minimalist, geometric precision

### The Elephant Reference
- **Integration**: Crescent curves subtly suggest elephant tusks
- **Purpose**: Nod to Hannibal's war elephants
- **Execution**: Sophisticated, abstract suggestion

## File Formats

### Vector Formats
- **SVG**: Scalable for all sizes
- **AI**: Adobe Illustrator for editing
- **EPS**: Encapsulated PostScript for print

### Raster Formats
- **PNG**: With transparency for web use
- **JPG**: For print applications
- **ICO**: Windows favicon format

## Implementation Notes

The logo design maintains the strategic intelligence and tactical precision that defines Barca-Strategos, while honoring the historical legacy of Hannibal Barca through sophisticated, modern design language.
