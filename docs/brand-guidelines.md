# Barca-Strategos Brand Guidelines

## 🐘⚔️ Brand Identity

### Core Brand Elements

**Name**: BARCA-STRATEGOS  
**Code Name**: BARCA-OS  
**Slogan**: "L'art de l'encerclement numérique"  
**Tagline**: "The Art of Digital Encirclement"

### Color Palette

#### Primary Colors
- **Rouge Carthaginois (Terracotta)**: `#E2725B` - Evokes the earth of Tunisia, blood of battles, and strategist's passion
- **Bronze Antique**: `#CD7F32` - Symbolizes robustness (Rust), historical armament, and longevity  
- **Noir Profond**: `#1A1A1A` - For "Black-Box" invisibility and technical elegance

#### Secondary Colors
- **Blanc Cassé**: `#F5F5F0` - For contrast and readability
- **Gris Technique**: `#4A5568` - For technical elements and code blocks

### Typography

#### Primary Font
- **Headings**: **Inter** or **Roboto** (Modern, technical, clean)
- **Body**: **Open Sans** or **Lato** (Readable, professional)

#### Font Hierarchy
```
H1: 2.5rem, Bold, Rouge Carthaginois
H2: 2rem, Bold, Bronze Antique  
H3: 1.5rem, Semi-Bold, Noir Profond
Body: 1rem, Regular, Noir Profond
Code: Monaco, Menlo, (Technical fonts)
```

## 🛡️ Logo Concept: "Le Piège de Cannes"

### Logo Design Principles

The logo embodies Hannibal's most famous military maneuver: the double envelopment at the Battle of Cannae.

### Logo Components

#### Primary Icon
- **Shape**: Two bronze crescents facing each other, creating an almost-closed circle
- **Symbolism**: Represents Numidian cavalry wings encircling the enemy
- **Color**: Bronze Antique with subtle gradient
- **Style**: Minimalist, modern, technical

#### Central Element  
- **Shape**: Small, minimalist data point or "key" in Rouge Carthaginois
- **Placement**: Center of the circle
- **Meaning**: Critical asset under protection
- **Style**: Clean, geometric

#### Subtle Elephant Reference
- **Integration**: Crescent curves subtly evoke elephant tusk curvature
- **Purpose**: Nod to Hannibal's war elephants without being literal
- **Execution**: Sophisticated, abstract suggestion

### Logo Variations

#### Full Logo
- Icon + "BARCA-STRATEGOS" text
- Text in custom font with Bronze Antique color
- Spacing and alignment precisely calculated

#### Icon Only
- Crescent design for app icons, favicons
- Scalable from 16px to 1024px
- Works on light and dark backgrounds

#### Monochrome Version
- Single color (Noir Profond)
- For security documents, technical specs
- Maintains recognizability

## 🎨 Visual Style Guide

### Design Philosophy
- **Tactical Precision**: Clean lines, geometric shapes
- **Strategic Intelligence**: Sophisticated, thoughtful composition
- **Modern Security**: Technical elegance with historical depth
- **Professional Authority**: Confident, expert positioning

### UI/UX Principles

#### Color Usage
- **Primary Actions**: Rouge Carthaginois (CTA buttons, alerts)
- **Secondary Elements**: Bronze Antique (headers, icons)
- **Background/Text**: Noir Profond and Blanc Cassé
- **Accent**: Gris Technique (borders, dividers)

#### Iconography
- **Style**: Line-based, minimalist
- **Weight**: 2px stroke, consistent
- **Meaning**: Each icon has tactical/security meaning
- **Set**: Complete icon library for all features

#### Layout Principles
- **Grid**: 8px baseline grid for precision
- **Spacing**: Consistent 16px/24px increments
- **Hierarchy**: Clear visual information architecture
- **Focus**: Strong contrast for accessibility

## 🎯 Implementation Guidelines

### Logo Usage Rules
- **Minimum Size**: 24px height for digital
- **Clear Space**: Equal to logo height on all sides
- **Background**: Always ensure contrast
- **Variations**: Use appropriate version for context

### Color Application
- **60% Primary**: Noir Profond for text, backgrounds
- **30% Secondary**: Bronze Antique for accents, highlights
- **10% Accent**: Rouge Carthaginois for actions, alerts

### Typography Rules
- **Hierarchy**: Maximum 3 heading levels
- **Readability**: Minimum contrast ratio 4.5:1
- **Consistency**: Same font family throughout
- **Accessibility**: Alt text for all meaningful images
