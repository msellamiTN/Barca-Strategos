## 🚀 Démarrage Rapide

### Prérequis

- Docker & Docker Compose
- 512MB RAM minimum
- 1GB d'espace disque

### Déploiement en Une Commande

```bash
# Cloner le repository
git clone https://github.com/barca-strategos/phoenix.git
cd phoenix

# Déployer Phoenix (configuration ultra-rapide)
curl -sSL https://deploy.barca-strategos.ai | bash

# Ou utiliser le script local
./phoenix-deploy.sh
```

### Accéder à Phoenix

- **Interface Web**: <http://localhost:3000>
- **API**: <http://localhost:8080>
- **Documentation API**: <http://localhost:8080/docs>
- **Bot Telegram**: Configurer le token dans `.env`

## 🛡️ Fonctionnalités de Sécurité

### Architecture Zero-Trust

- **Authentification Mutuelle**: Tous les agents se vérifient mutuellement
- **Communication Chiffrée**: Chiffrement de bout en bout pour tous les messages
- **Exécution Isolée**: Environnements isolés pour les opérations des agents
- **Journalisation d'Audit**: Suivi complet des événements de sécurité

### Protection contre les Injections de Prompt

Protection avancée contre:

- Remplacements de prompt système
